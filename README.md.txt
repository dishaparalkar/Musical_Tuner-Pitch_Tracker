# 🎵 Musical Tuner & Pitch Tracker

A real-time Musical Tuner and Pitch Tracker developed using Python, Digital Signal Processing (DSP), FFT, and Streamlit.

## Project Overview

This application captures live microphone audio and performs real-time pitch detection. The detected frequency is mapped to the nearest musical note and displayed along with tuning information.

## Features

- Real-time microphone input
- FFT-based frequency analysis
- Peak frequency detection
- Musical note mapping
- Cents error calculation
- Sharp / Flat / In Tune indication
- Waveform visualization
- FFT Spectrum visualization
- Pitch trend analysis
- Interactive Streamlit dashboard

## Technologies Used

- Python
- NumPy
- Streamlit
- SoundDevice
- Plotly
- Matplotlib

## DSP Concepts Used

- Audio Sampling
- Hamming Window
- Fast Fourier Transform (FFT)
- Peak Detection
- Frequency Smoothing
- Musical Note Mapping
- Cents Error Calculation
